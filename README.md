# 🔥 FireSafe - Comprehensive Fire Safety & Awareness Website

A professional, comprehensive website dedicated to fire safety education, prevention, emergency response, and wildfire awareness. Designed to inform and protect communities while emphasizing the importance of preventing fires.

## 📋 Overview

FireSafe is a responsive, user-friendly website that provides:

- **Fire Science & Information** - Educational content about how fires work
- **Fire Prevention Tips** - Practical safety guidelines for home, kitchen, electrical safety, and more
- **Emergency Services** - Information about fire departments and how to respond to emergencies
- **Wildfire Awareness** - Prevention strategies and evacuation preparation
- **Community Resources** - Links to trusted fire safety organizations and training programs

## 🎯 Purpose

**Don't Start Fires - Respect Fire's Power**

This website aims to:
1. Educate the public about fire safety
2. Reduce fire-related incidents through prevention
3. Prepare communities for fire emergencies
4. Raise awareness about wildfire threats
5. Build connections between communities and local fire departments

## 🔧 Customization

### Colors
To change the color scheme, modify the CSS variables in `styles.css`:

```css
:root {
    --primary-color: #ff6b35;
    --secondary-color: #f7931e;
    --dark-color: #1a1a1a;
    --light-color: #f5f5f5;
    --text-color: #333;
    --warning-color: #e74c3c;
}
```

### Content
Edit the HTML content in `index.html`. Each section uses semantic structure for easy editing.

### Animations
Modify animations in `styles.css` (look for `@keyframes`) or adjust timing in `script.js`.
